## Purpose of Source Availability

Essential has made the source code available to:

1. **Enable transparency** - Demonstrate commitment to the community by making code visible
2. **Allow verification** - Let users confirm that distributed binaries match the published source
3. **Support auditing** - Enable security researchers and users to audit the code
4. **Build trust** - Show exactly what the mod does without hiding behind closed source
