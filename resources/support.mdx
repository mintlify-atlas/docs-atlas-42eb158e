Need help with Essential? There are several ways to get support, report issues, and connect with the Essential community.

## Discord Community

The primary support channel for Essential is the **official Discord server**:

**[Join Essential Discord](https://essential.gg/discord)**

The Discord server offers:

- **Dedicated support channels** - Get help from the community and Essential team
- **Real-time assistance** - Quick responses to questions
- **Community discussions** - Connect with other Essential users
- **Release announcements** - Stay updated on new versions
- **Feature discussions** - Learn about upcoming features

<Tip>
  For repository-specific questions or assistance, use the dedicated channels in the Discord server.
</Tip>

## GitHub Issues

For bug reports, feature requests, and technical issues related to the source code:

**[Essential GitHub Repository](https://github.com/EssentialGG/Essential/issues)**

### When to Use GitHub Issues

- **Build problems** - Issues compiling from source
- **Verification failures** - Checksum mismatches or build discrepancies
- **Source code bugs** - Issues found when reviewing the code
- **Documentation errors** - Corrections to technical documentation

### Issue Guidelines

When opening an issue:

1. **Search existing issues** - Check if your issue is already reported
2. **Provide details** - Include version numbers, error messages, and reproduction steps
3. **Be specific** - Clearly describe the problem and expected behavior
4. **Include logs** - Attach relevant log files and stack traces
5. **Minecraft version** - Specify which version(s) are affected

<Warning>
  GitHub Issues are for **source code and build-related issues only**. For general mod support, use Discord instead.
</Warning>

## Legal and Security Contacts

### License Questions

For questions about the source-available license:

**legal@essential.gg**

Contact legal for:
- License interpretation
- Usage permission inquiries
- Verification discrepancies
- Legal compliance questions

### Security Issues

If you discover a security vulnerability:

<Warning>
  **Do not open public GitHub issues for security vulnerabilities.** Report them privately to the Essential team.
</Warning>

1. Contact Essential through Discord
2. Email security-sensitive issues to official channels
3. Wait for confirmation before public disclosure

When seeking support:

## Contributing Back

While the license restricts code contributions, you can help the community by:

- **Answering questions** in Discord
- **Documenting solutions** to common problems
- **Testing releases** and providing feedback
- **Reporting issues** clearly and thoroughly
- **Verifying checksums** and reporting discrepancies

<Tip>
  The Essential community is strongest when users help each other. Consider sharing your knowledge in Discord once you've solved a problem.
</Tip>

## What to Include in Support Requests

For the fastest help, include:

- **Essential version** - Check in-game or jar filename
- **Minecraft version** - e.g., 1.21.8, 1.16.5, etc.
- **Mod loader** - Forge or Fabric (and version)
- **Operating system** - Windows, macOS, or Linux
- **Java version** - Output of `java -version`
- **Other mods** - List installed mods if relevant
- **Error messages** - Full error text or logs
- **Reproduction steps** - How to recreate the issue
- **Expected behavior** - What should happen
- **Actual behavior** - What actually happens
